description: 'commands of the info section'
---


### github

use `/github repo <query>` replace "<query>" with the repo's name to get the information about it.

use `/github user <user>`  replace "<user>" to get information about a github user.


### define

use `/define <word>" replace "<word>" with the word you want to search.

use `/urban-directory <term>` replace "<term>" with the term you want to search.
